# Search And Replace

##


## Usage
```
Usage:
  search-and-replace [OPTIONS] Search Replace

Application Options:
  -d, --dry-run  Do not change anything
  -r, --regexp   Treat search string as regular
                 expression
  -v, --verbose  Show verbose debug information

Help Options:
  -h, --help     Show this help message

Arguments:
  Search
  Replace
```
